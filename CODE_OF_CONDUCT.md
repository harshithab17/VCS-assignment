# Contributor Code of Conduct

Thank you for contributing to this project!

## Expectations
All contributors are expected to:
- Be respectful and kind in communication.
- Use appropriate language.
- Provide helpful and constructive feedback.
- Help create a positive and welcoming environment.

## Unacceptable Behavior
The following will not be tolerated:
- Harassment or discrimination
- Personal attacks or rude comments
- Sharing harmful or incorrect information intentionally

## Reporting Issues
If you experience any unacceptable behavior, please open an issue in this repository.

## Enforcement
Project maintainers may warn or remove contributors who violate this Code of Conduct.

---

By contributing to this repository, you agree to follow this code of conduct.
